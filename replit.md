# Portfolio Website - Mrinal Kalita

## Overview
This is a personal portfolio website for Mrinal Kalita, a Software Engineer and Data Analyst. Built with React and styled with Tailwind CSS.

## Project Structure
- `ViewSide/my-app/` - Main React application
  - `src/` - Source code
    - `pages/` - Main page components
    - `mobilepages/` - Mobile-responsive page components
    - `componets/` - Reusable UI components
    - `Assets/` - Images and logos
  - `public/` - Static assets
  - `build/` - Production build output

## Technologies
- React 19
- Tailwind CSS
- FontAwesome icons
- EmailJS for contact form
- react-scroll for smooth scrolling

## Development
The app runs on port 5000 with the command:
```
cd ViewSide/my-app && npm start
```

## Environment Variables
- `PORT=5000` - Development server port
- `HOST=0.0.0.0` - Bind to all interfaces
- `DANGEROUSLY_DISABLE_HOST_CHECK=true` - Allow proxy access

## Features
- Responsive design (desktop and mobile)
- Project showcase
- Testimonials section
- Contact form with EmailJS integration
- Skills/tools section
